Garden Party

A simple gardening game catered towards younger age groups to practice adding and subtracting. For each level, the user must reach a given score goal before time runs out. This was my first project, built in a 1 week sprint using HTML, CSS, Bootstrap, Javascript, and jQuery for DOM manipulation.

Ascend LA

Ascend LA is a database of hikes in Los Angeles, CA where users can create an account, search hikes, post reviews, and photos in their personal gallery. It utilizes the Googlemaps API to view trail locations. This app was built using Ruby on Rails and data models were built on PostgresSQL.

Le Closet

Shop online and search products from your favorite retailers all in one place. A user can conveniently create a virtual shopping cart, or "closet", to save products and make purchases. As Project Lead, I managed team collaboration using Git, GitHub and team work-flow. This app was build in Node.js and utilized MongoDB database. We used the Shopstyle API along with asynchronous AJAX calls and Google OAuth for authentification.
